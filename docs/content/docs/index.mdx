## ğŸš€ Quick Start

Get started with OpenHooks in under a minute:

### 1. Install the CLI

```bash
npm install -g open-hook
```

### 2. Initialize Your Project

```bash
open-hook init
```

### 3. Add Your First Hook

```bash
open-hook add useClickOutside
```

### 4. Use in Your Component

```tsx
import { useRef, useState } from "react";
import { useClickOutside } from "./hooks/useClickOutside";

function MyComponent() {
  const ref = useRef(null);
  const [isOpen, setIsOpen] = useState(false);

  useClickOutside(ref, () => setIsOpen(false));

  return <div ref={ref}>{/* Your component content */}</div>;
}
```

<Callout type="tip">
  Want to see all available hooks? Run `open-hook list` to browse the complete
  collection.
</Callout>
